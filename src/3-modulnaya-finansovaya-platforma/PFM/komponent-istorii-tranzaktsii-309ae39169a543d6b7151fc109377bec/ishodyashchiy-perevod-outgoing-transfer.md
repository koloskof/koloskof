# Исходящий перевод (Outgoing Transfer)

## Общая информация

- Статус
- Дата и Время
- Название платежа
- Сумма
- Описание
- Комментарий

## Действия

- Убрать из избранного
- Написать в чат
- Получить чек

## Детали операции

- Номер для поддержки
- Кошелёк получателя
- Телефон получателя
- Почта получателя
